# Meadowsweet

Meadowsweet allows the creation of web application using an action-oriented
pattern. It is implemented on top of Ada Servlet.

This repository contains a simple example CRUD application in the
subdirectory samples/phonebook.

For generating a response using a template engine, a renderer library is
required.
[Meadowsweet_Jintp](https://github.com/rehartmann/meadowsweet_jintp)
provides a renderer which uses the
[JinTP](https://github.com/rehartmann/jintp) template engine.
